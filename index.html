To connect to your computer from another one using VNC open Finder and press ⌘+k. In the Server Address enter vnc:// followed by your computer name or IP address, for example: vnc:127.0.0.1:5901.

互联网骨干网全面解析

骨干网一词源自NSFNet，这是一种用于早期研究的网络，该网络由美国国家科学基金会(NSF)出资兴建。它创建了至今仍在使用的分层结构模型。这种模型中，本地服务提供商连接到区域服务，而后者又依次连接到全国或全球的服务提供商。目前，已有许多骨干网相互连接在一起，这就使得任何两台主机之间都可通信。此外，许多区域性的网络避开了骨干网而直接彼此相连。

对等互联所采取的形式多种多样，有的是在公共的NAP处连接，有的则是通过直联的方式。但是对于绝大部分互联网骨干运营商来说，单纯依靠对等互联往往不能满足要求，而必须采用对等互联与非对称互联相结合的方式。

CHINANET八大节点：

中国电信CHINANET骨干网核心层由北京、上海、广州、沈阳、南京、武汉、成都、西安等8个城市的核心节点组成。核心节点之间为不完全网状结构。以北京、上海、广州为中心的三中心结构，其他核心节点分别以至少两条高速ATM链路与这三个中心相连。另外各省还建立了二级节点。

Python的内存管理模型

对于任何编程语言，我们都需要从下面两个方面去理解其内存管理(memory management)模型: * 内存的分配 * 无用内存的回收 (手动内存回收, 如C语言，以及自动内存回收，如Java/Python)

Python memory is managed by Python private heap space. Python语言中的内存管理是通过私有堆空间(private heap space) 进行的。任何Python对象以及数据结果都是存储在一个私有的堆空间(private heap spaace)中。程序员必须通过Python解释器进行相关的操作，无法直接访问该私有堆空间。

Python的垃圾回收机制: 在Python中，每个对象都有指向该对象的引用总数---引用计数(refcount). 对于任何一个Python对象而言，以下操作会导致引用计数的增减: 创建对象，被赋值给其他对象，该对象成为别的容器对象中的一个元素，被作为参数传递给函数. 导致引用计数减小的操作: * 对象的别名被显式的销毁 如 del m * 对象的一个别名被赋值给其他对象 * 对象从一个窗口对象中移除，或，窗口对象本身被销毁 a.remove(123) * 一个本地引用离开了它的作用域，比如上面的foo(x)函数结束时，x指向的对象引用减1。

当Python的某个对象的引用计数降为0时，说明没有任何引用指向该对象，该对象就成为要被回收的垃圾。
