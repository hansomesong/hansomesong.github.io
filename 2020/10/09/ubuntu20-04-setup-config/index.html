Virtualbox 屏幕分辨率

一、免密sudo

为了避免每次使用 sudo 命令时都输入密码，我们可以将密码关闭。操作方法： 第一步：终端输入命令sudo visudo，打开 visudo； 第二步：找到 %sudo ALL=(ALL:ALL) ALL 这一行修改为%sudo ALL=(ALL:ALL) NOPASSWD:ALL

二、更换软件源

Ubuntu 默认的软件源是境外的，速度上会有些问题，我们可以在「Software & Updates」(软件和更新)中选择国内的镜像。

5. 安装 Python2

Ubuntu20.04 自带了 python3，但是某些第三方工具或者脚本还是用的 python2，因此我们需要自行安装

apt install python

6. 安装 Git

7. 中文输入法

由于搜狗输入法目前还不支持Ubuntu20.04，加之其对高分屏支持不太友好，因此不再折腾选择了 ibus-libpinyin。

复制代码接着在应用程序中找到「Language Support」(语言支持)，更改「Keyboard input method system」(键盘输入法系统)为「IBUS」。重启系统，然后在Settings>Region & Language>Input Sources（设置>区域与语言>输入源）中新增「Chinese(Intelligent Pinyin)」(中文(智能拼音))就可以使用中文输入法了。

8. proxychains-ng 安装

首先一定要确保gcc automake make之类的工具安装了。不然在源码代码编译的时候，./configure命令会出现错误结果。

然后切换到/opt目录之下，执行 git clone

9. Clash for Ubuntu

第一步： 下载适合的 Clash 二进制文件，一般个人的 64 位电脑下载 clash-linux-amd64-v0.17.1.gz 即可 GitHub 项目地址:https://github.com/Dreamacro/clash/releases
